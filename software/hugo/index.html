Site setup

Install the EXTENDED VERSION to enable customization of the Hermit theme!

Commands:

  • hugo new site soundsbrown
  • cd soundsbrown
  • git init
  • git submodule add https://github.com/Track3/hermit.git themes/hermit
  • echo theme = “hermit” >> config.toml
  • hugo new posts\hello.md
  • hugo
  • hugo server -D

Some settings need to be copied from themes/hermit/exampleSite/config.toml to your own config.toml
