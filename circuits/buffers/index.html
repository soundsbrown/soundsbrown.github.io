Application notes

Supply voltage is important:

  • All tested amps max out at somewhere between 3V and 4V when using a 5V supply. Use 12V and put other components (e.g. Arduino) behind a regulator!
  • When using unipolar (0-12V) supply:
    • LM741 floors at 2V or so – anything below that gets clamped.
    • TL07x inverts below 1.5V or so (i.e. X turns into 12V - X when X < 1.5V)
    • TL08x and LM32x both seem very reliable, for CV as well as audio
